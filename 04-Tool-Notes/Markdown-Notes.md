# Markdown Syntax Cheat Sheet

| Element | Syntax | Result / Description |
| :--- | :--- | :--- |
| **Heading 1** | `# Text` | Main Title (H1) |
| **Heading 2** | `## Text` | Section Heading (H2) |
| **Heading 3** | `### Text` | Sub-section Heading (H3) |
| **Bold** | `**Text**` | **Strong** emphasis |
| **Italic** | `*Text*` | *Light* emphasis |
| **Blockquote** | `> Text` | Used for NIST/Policy quotes |
| **Inline Code** | `` `Text` `` | Highlighted `technical` terms |
| **Code Block** | \`\`\`bash ... \`\`\` | Multi-line terminal commands |
| **Unordered List** | `* Text` or `- Text` | Bullet points |
| **Ordered List** | `1. Text` | Numbered steps |
| **Task List (Empty)** | `- [ ] Text` | Unchecked compliance item |
| **Task List (Done)** | `- [x] Text` | Completed compliance item |
| **Horizontal Rule** | `---` | Section divider line |
| **External Link** | `[Name](URL)` | Clickable web link |
| **Internal Link** | `[Name](./path/file.md)` | Link to another note in your repo |
| **Image** | `![Alt](path/image.png)` | Displays mind map or screenshot |
| **Table** | `| Col | Col |` | Organized data/Control matrices |
| **Strikethrough** | `~~Text~~` | ~~Redacted~~ or obsolete info |
| **Superscript** | `Text^2` | Mathematical notation |
| **Subscript** | `Text~2` | Chemical/Technical notation |
| **Footnote** | `[^1]` | Reference link at bottom of page |
| **Math (Inline)** | `$Equation$` | LaTeX for Risk formulas |
| **Math (Block)** | `$$Equation$$` | Centered large scale formulas |
